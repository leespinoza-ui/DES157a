Set 1 — Bows

Soft nude base with delicate white bow detailing.

  • Small red accents
  • Glossy finish

This set feels soft and romantic to me. I wanted something that felt delicate but still detailed, like a Valentine’s card. The bows represent the sweeter, more playful side of my personality.

Set 2 — Red Tips

Bold red framing the nail for a dramatic look.

  • Red French tip framing
  • Heart + rhinestone details
  • Clean, sharp edges

This design feels more bold and confident. I love how the red frames the nail and makes everything stand out. It reminds me of dressing up for Valentine’s Day and wanting something dramatic but still clean.

Set 3 — Valentine Mix

Mix-and-match hearts, gems, and layered designs.

  • 3D gem accent nail
  • Red + nude layers
  • Statement nail focus

This set was the most fun to design because I mixed everything I love hearts, gems, layering, and dimension. It feels expressive and playful, like a collage of Valentine’s emotions all in one.
