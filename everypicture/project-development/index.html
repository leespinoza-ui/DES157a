Every Picture — Project Development

Conceptual Changes

  • Shifted from showing nail sets as a simple collection to framing them as personal narrative fragments.
  • Focused more on process and meaning, not just finished results.
  • Considering each nail set as a “chapter” rather than a single image.
  • Planning to use minimal text, revealed gradually, to let the images lead the storytelling.
  • Exploring how color, shape, and texture communicate emotion without explanation.

Design Revisions

  • Simplified layout so images are the main focus.
  • Limited paragraph width for readability (no centered or justified text).
  • Increased spacing between images to slow down viewing and encourage close looking.
  • Using a clean, neutral background so colors and details of the nails stand out.
  • Consistent image sizing and alignment to strengthen visual hierarchy.
  • Typography kept simple to avoid competing with the imagery.
